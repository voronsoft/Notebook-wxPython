tuple:КОРТЕЖИ - Неизменяемый тип данных / Упорядоченная коллекция элементов различных типов.
count(x):Возвращает количество вхождений элемента X в кортеже.:No example
index(x):Возвращает индекс первого вхождения элемента X в кортеже:No example
len(x):Возвращает длину (количество элементов) кортежа:No example
(1, 2) + (3, 4):Поддерживает конкатенацию (сложение):No example
(3, 4) * 3:Поддерживает повторение:(3, 4) * 3 результат (3, 4, 3, 4, 3, 4)
a = tuple():Создать кортеж:No example
a = (1,2):Создать кортеж:No example
a, b = (1, 2):Поддерживает распаковку:a = 1 , b = 2
my_tuple = (5,):Создать кортеж, в конце запятая:isinstance(my_tuple, tuple) > True
