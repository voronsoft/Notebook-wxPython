list:СПИСКИ - Изменяемый тип данных / Упорядоченная коллекция данных.
list.append(x):Добавляет элемент в конец списка:No example
list.extend(L):Расширяет список list, добавляя в конец все элементы списка L:No example
list.insert(i, x):Вставляет на i-ый элемент значение x:No example
list.remove(x):Удаляет первый элемент в списке, имеющий значение x. ValueError, если такого элемента не существует:No example
list.pop(i):Удаляет i-ый элемент и возвращает его. Если индекс не указан, удаляется последний элемент:No example
list.index(x, [start [, end]]):Возвращает положение первого элемента со значением x (при этом поиск ведется от start до end):No example
list.count(x):Возвращает количество элементов со значением x:No example
list.sort([key=функция]):Сортирует список на основе функции:No example
list.reverse():Разворачивает список:No example
list.copy():Поверхностная копия списка:No example
list.clear():Очищает список:No example
