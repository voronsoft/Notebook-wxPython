Приложение "Блокнот"
Создает вкладки в которых содержатся записи.
Разрабатывалось как шпаргалка команд языков программирования.
Есть встроенная БД (база данных) языка Python.

Поддерживает функции:
-- Логирование
-- Импорт данных в файл json
-- Экспорт данных в файл(ы) json
-- Поиск данных
-- Удаление записей
-- Добавление записей
-- Изменение записей
-- Документация с изображениями
-- Просмотр файлов PDF
